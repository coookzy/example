# Slop Bot Assets

This folder contains all images and media assets for the Slop Bot website.

## Required Assets

### Logo
- `slop-bot-logo.png` - Main Slop Bot logo
  - Style: Terminal aesthetic, matrix green, monospace font
  - Format: PNG with transparency
  - Size: 512x512px recommended

### Open Graph / Social Media
- `slop-bot-og.png` - Social media preview image (referenced in index.html)
  - Format: PNG
  - Size: 1200x630px (Open Graph standard)
  - Content: Slop Bot branding with tagline

- `scan-lines.png` - Optional CRT scan line texture

## Design Guidelines

**Color Palette:**
- Matrix Green: #00ff41
- Terminal Cyan: #00d9ff
- Error Red: #ff3366
- Terminal Yellow: #ffd700
- Glitch Purple: #b537ff

**Fonts:**
- IBM Plex Mono
- VT323
- Major Mono Display
- Source Code Pro

**Style:**
- Terminal/command-line aesthetic
- CRT monitor vibes (scan lines, vignette)
- Matrix/hacker aesthetic (NOT Win95)
- Self-aware content generator theme

## Notes

Images are not required for the site to function - all visual effects are CSS-based. Assets are for branding and social media only.
